## Doctype og head

```html
<!doctype html>
  <html lang="en">
    <head>
      <meta charset="utf-8">
      <!-- Instruct Internet Explorer to use its latest rendering engine -->
      <meta http-equiv="x-ua-compatible" content="ie=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <title>Tittel</title>
    </head>
```
